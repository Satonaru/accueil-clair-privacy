Politique de confidentialité — Accueil Clair

Dernière mise à jour : 07 juillet 2026

Accueil Clair est une application locale destinée à aider la propriétaire d’un compte Instagram professionnel à gérer ses nouveaux abonnés, ses messages de bienvenue, ses réponses reçues et ses relances.

Données traitées

L’application peut traiter les données suivantes lorsque l’utilisateur connecte son compte Instagram professionnel via les outils officiels de Meta :

  • identifiant du compte Instagram professionnel ;
  • nom du compte ;
  • nom d’utilisateur Instagram ;
  • nombre d’abonnés ;
  • nombre de publications ;
  • conversations Instagram autorisées par l’API Meta, si l’accès messagerie est accordé ;
  • contenu des messages Instagram autorisés par l’API Meta ;
  • statuts internes ajoutés par l’utilisateur : contacté, réponse reçue, relance prévue.

Finalité du traitement

Ces données sont utilisées uniquement pour :

  • afficher les statistiques du compte Instagram connecté ;
  • aider la propriétaire du compte à suivre les nouveaux abonnés ;
  • préparer ou envoyer des messages de bienvenue lorsque cela est autorisé ;
  • afficher les conversations Instagram autorisées ;
  • suivre les réponses reçues et les relances ;
  • améliorer l’organisation locale du travail de la créatrice de contenu.

Stockage des données

Accueil Clair est une application locale. Les données sont stockées sur l’ordinateur de l’utilisateur. Elles ne sont pas revendues, partagées ou transférées à des tiers.

Les tokens d’accès Meta peuvent être mémorisés localement sur l’ordinateur de l’utilisateur afin d’éviter une reconnexion manuelle à chaque utilisation. L’utilisateur peut supprimer ce token depuis l’application.

Partage des données

Accueil Clair ne vend pas les données personnelles, ne les partage pas avec des annonceurs et ne les utilise pas à des fins de profilage publicitaire.

Suppression des données

L’utilisateur peut supprimer les données stockées dans Accueil Clair depuis l’application ou en supprimant les fichiers locaux de l’application.

Pour demander la suppression des données ou obtenir de l’aide, l’utilisateur peut contacter :
sophro.alabi@gmail.com

Sécurité

Accueil Clair utilise les API officielles de Meta lorsque l’utilisateur connecte son compte. L’application ne demande pas le mot de passe Instagram et ne contourne pas les systèmes de sécurité de Meta.
